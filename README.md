# Smart Light Switch with Fallback for Home Assistant

This repository provides step-by-step guides for converting a traditional electrical control into a smart and reliable lighting setup using Home Assistant, a Shelly 1 Mini, and a smart bulb.  
A built-in fallback mechanism ensures the light remains usable even if Home Assistant is offline.
